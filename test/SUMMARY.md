## 💡 نصائح مهمة

### ✨ نصيحة 1: استخدم Environment في Postman

```
بدلاً من:
http://localhost:5000/api/courses

استخدم:
{{BASE_URL}}/api/courses
```

### ✨ نصيحة 2: احفظ Tokens تلقائياً

في tab "Tests" لطلب Login، أضف:

```javascript
if (pm.response.code === 200) {
    var jsonData = pm.response.json();
    pm.environment.set("STUDENT_TOKEN", jsonData.token);
}
```

### ✨ نصيحة 3: استخدم Collection Runner

```
Collection → Run → اختر الطلبات → Run
```

### ✨ نصيحة 4: راجع الأخطاء الشائعة

- ❌ **401**: Token منتهي → Login مرة أخرى
- ❌ **403**: Token خاطئ → استخدم الـ token الصحيح
- ❌ **400**: بيانات خاطئة → راجع الـ Body
- ❌ **429**: طلبات كثيرة → انتظر 15 دقيقة
