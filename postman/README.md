# Postman API Collection

## الملفات

- **`collection.json`** - مجموعة Postman Collection كاملة لجميع API endpoints

## الاستيراد

1. Postman → Import → Upload Files
2. اختر `postman/collection.json`
3. Import

## التوثيق الكامل

راجع [docs/API/POSTMAN.md](../docs/API/POSTMAN.md) للتفاصيل الكاملة:
- المتغيرات (BASE_URL, ADMIN_TOKEN, إلخ)
- تسجيل الدخول (identifier + password)
- قائمة جميع الـ endpoints
- استكشاف الأخطاء
